Note:

  • If you have books in multiple locations you can use the location field to specify the location of the books

  • To add an EBook, type "E-Book" in the location Field, Enter the filename in the E-Book location field (example.pdf), and add the file in the server in the following location 'static/files/', if the files dir is not there create one with the same name Case Sensitive.

  • To add a cover image for a book, you can either:
    • 1. Upload the image to the server in the 'static/images/books' (if the books dir is not there create one with the same name Case Sensitive) and provide the relative path in the cover image field (e.g., '../static/images/books/example.png').
    • 2. Use an external image URL and provide the full URL in the cover image field.
